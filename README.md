# cse412-photoshare
a photo sharing social media app

# Client (Front-end)
To start the client
1. cd into client directory  
2. install dependencies: ```npm i```  
3. start client: ```npm start```  

# Server (Back-end)
To start the server
1. cd into server directory  
2. install dependencies: ```npm i```  
3. start server: ```npm run dev```
