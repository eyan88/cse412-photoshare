# cse412-photoshare
a photo sharing social media app

# Client (Front-end)
To start the client
1. cd into client directory  
2. run ```npm i```to install dependencies  
3. run ```npm start```to start  

# Server (Back-end)
To start the server
1. cd into server directory  
2. run ```npm i``` to install dependencies  
3. run ```npm run dev```
